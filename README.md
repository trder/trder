# trder:开源量化交易平台

> 开源量化交易平台。

一个能够创建、测试、运行量化交易系统以及训练量化交易AI的集成开发平台。

项目地址：https://github.com/trder/trder/

## 下载

把整个工程源码下载到本地，解压即可。

## 环境准备

需要python 3.10以上环境。

## 安装依赖包

> pip install -r requirements.txt

## 测试量化交易系统

> cd trder
> 
> python trder.py 交易系统名

## 运行量化交易系统

> cd trder
> 
> python run.py 交易系统名
